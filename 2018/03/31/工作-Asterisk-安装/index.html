Asterisk

2018年03月31日

Asterisk 是开源电话系统软件，内置软件源中。 可通过源码编译，在编译的过程中首先需要安装编译依赖然后编译菜单选择。 这个过程十分繁琐，可以直接通过 apt install asterisk 安装. 且版本也不会太老。 asterisk 缺少一个可用的 UI. 如果是 FreeSwitch 可以尝试 YouPBX. 若只是需要测试电话系统， mac 可以直接通过 brew 安装 FreeSwitch 或 opensips.

通过树莓派安装 Asterisk 是一个快速的方式。
