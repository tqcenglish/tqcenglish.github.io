# web 框架介绍
## [relay](https://github.com/facebook/relay)

与查询一起使用。
## [meteor](https://github.com/meteor/meteor/)

适合实时引用，对 App 接口不友好。
## [phoneix](http://www.phoenixframework.org/)

适合游戏的后端框架。
## [sails](http://sailsjs.org/)

MVC 框架，Express 版本落后。
