Egg 与 KOA 对比

EGG 与 KOA 比较, EGG 基于 KOA, 有更多的 MVC 限制(也可以理解为方便)。 koa 只是基础框架，一个完整的 http 至少包括如下部分

  • 静态目录挂载 kao-static
  • 路由请求 koa-router
  • 参数解析 koa-bodyparser

另外 EGG 框架支持多进程, 效率更高。
